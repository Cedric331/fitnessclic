# Captures d'écran de l'interface

Placez vos captures d'écran de l'interface dans ce dossier.

## Fichiers attendus :

1. `screenshot-creation.png` - Capture de la page de création de séance
2. `screenshot-library.png` - Capture de la bibliothèque d'exercices
3. `screenshot-clients.png` - Capture de la gestion des clients
4. `screenshot-dashboard.png` - Capture du dashboard principal

## Recommandations :

- Format : PNG ou JPG
- Taille recommandée : 1920x1080px ou supérieure
- Ratio : 16:9 (format paysage)
- Poids : Optimisez les images pour le web (max 500KB par image)

Les images seront automatiquement affichées dans la section "Découvrez l'interface" de la page d'accueil.
